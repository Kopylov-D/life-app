# Life App

Приложение на стеке MERN, состоящее из двух модулей:

- учет расходов и доходов.
- планировщик дел с декомпозицией задач и целей.

Демонстрационная версия - [Life App](https://life-up.herokuapp.com/)

## Используется:

Typescript (клиент и сервер).

**Frontend:**

- ReactJS, Redux, Reselect, React Router, React hooks, Thunk, Recharts, Axios, SCSS.

**Backend:**

- NodeJS, Express, Mongoose, JWT.

## Реализовано:

- Регистрация и авторизация (JWT)
- Работа с базой данных MongoDB
- Фильтрация данных на сервере
- Роутинг
- Управление состоянием с помощью redux
- Система UI-компонентов

## Установка

1. `git clone https://github.com/Kopylov-D/life-app.git`.
2. `npm install`.
3. `npm run client:install`.
4. `npm run dev`.

Открыть http://localhost:3008 в браузере.
